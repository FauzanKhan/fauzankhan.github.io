I'm Fauzan, a self taught web developer working on cutting edge technologies currently specializing in Data Visualization. I'm a detail oriented Front End Developer who loves to implement elegant and high performance UIs, brainstorm on designs for applications and I believe that design is about communicating to users rather than just making something look nice.

Since the beginning of my journey in IT, I've had a clear and focused goal - to learn as much as I can, keep pushing myself, and make things that I can take pride in. I've learned everything I know so far all by myself from Front End Technologies such as HTML5, CSS3, JavaScript, Angular, D3, jQuery, Bootstrap, Foundation etc to Back End and System programming languages like C, Java, Ruby, & python.

I love what I do and I like reading stuff about web development and I enjoy incorporating the stuff I read about in my workflow. I work real hard untill I can get to the point from where I can work smart. I always try to build/lookout for stuff that can make my development workflow easier.
